# up.sh

Скрипт для автоматического создания директории с текущей датой и обновления симлинка `latest`.

## Описание

Скрипт создает директорию с форматом `YYYY-MM-DD` (например, `2024-01-15`) и обновляет симлинк, который всегда указывает на директорию сегодняшнего дня. Это удобно для организации файлов по датам.

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/bogdanovby/up.sh.git
cd up.sh

# Сделать скрипт исполняемым
chmod +x up.sh

# Опционально: добавить в PATH или создать симлинк
sudo ln -s $(pwd)/up.sh /usr/local/bin/up
```

## Использование

### Базовое использование

Просто запустите скрипт:

```bash
./up.sh
```

Что произойдет:
- Создастся директория `/home/pavel/tmp/2024-01-15` (с текущей датой)
- Обновится симлинк `/home/pavel/latest` → `/home/pavel/tmp/2024-01-15`

### Примеры вывода

```bash
$ ./up.sh
Создание директории: /home/pavel/tmp/2024-01-15
Обновление симлинка /home/pavel/latest -> /home/pavel/tmp/2024-01-15
Готово! Директория создана и симлинк обновлен.
```

### Кастомизация путей через переменные окружения

Вы можете переопределить пути:

```bash
# Изменить базовую директорию
BASE_DIR=/tmp/backups ./up.sh

# Изменить имя симлинка
LATEST_LINK=/home/pavel/today ./up.sh

# Оба сразу
BASE_DIR=/var/log/daily LATEST_LINK=/var/log/today ./up.sh
```

### Практические примеры

#### 1. Быстрый переход в директорию сегодняшнего дня

```bash
./up.sh
cd /home/pavel/latest
# или просто
cd ~/latest
```

#### 2. Организация бэкапов по датам

```bash
BASE_DIR=/backup/daily ./up.sh
cp important_file.txt /backup/daily/latest/
# или
cp important_file.txt $(readlink -f /backup/daily/latest)/
```

#### 3. Логирование в директорию дня

```bash
BASE_DIR=/var/log/app ./up.sh
echo "Log entry" >> /var/log/app/latest/app.log
```

#### 4. Ежедневные заметки

```bash
BASE_DIR=~/notes ./up.sh
vim ~/latest/daily-notes.md
```

#### 5. Использование в cron для автоматического создания директорий

```bash
# В crontab
0 0 * * * /home/pavel/src/up.sh/up.sh
```

Это создаст новую директорию каждый день в полночь.

#### 6. Скрипт-обертка для удобства

Создайте алиас в `~/.bashrc`:

```bash
alias today='~/src/up.sh/up.sh && cd $(readlink -f ~/latest)'
```

Тогда можно просто:
```bash
today  # создаст директорию и перейдет в неё
```

## Структура после выполнения

```
/home/pavel/
├── tmp/
│   ├── 2024-01-15/    ← директория сегодняшнего дня
│   ├── 2024-01-14/    ← вчерашняя
│   └── 2024-01-13/    ← позавчерашняя
└── latest -> tmp/2024-01-15/  ← симлинк на сегодня
```
